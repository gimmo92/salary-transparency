# Analisi Trasparenza Salariale

Applicazione Vue.js per l'analisi e la visualizzazione della trasparenza retributiva.

## Setup

```bash
npm install
```

## Sviluppo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview build

```bash
npm run preview
```
